## the way to go

#### go

> Go是一种开放源代码编程语言，可轻松构建简单，可靠且高效的软件。21世纪C语言

- 语法简洁。类C语言风格，简单易学，学习曲线平缓
- 代码风格统一。Go 语言提供了一套格式化工具——go fmt
- 开发效率高。开发效率与执行效率的完美结合，像写Python代码效率，编写C代码性能
